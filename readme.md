# 🌌 SkyWander.Api

**SkyWander.Api** is a **Python application** designed to control an **equatorial telescope mount** through integration with **Stellarium**, enabling automated pointing, precise movement, and programmatic control of astronomical equipment.

The project acts as an intermediary layer between planetarium software and telescope hardware, making observation automation possible.

---

## 🚀 Purpose

The main goal of **SkyWander.Api** is to allow a telescope mounted on a **motorized equatorial mount** to be controlled via software, using **Stellarium** as the visual interface for selecting celestial objects.

With this system you can:

- 🔭 Automatically point the telescope to stars, planets, and deep-sky objects  
- 🌍 Synchronize astronomical coordinates (RA/DEC)  
- 🧭 Control mount movements  
- 🧠 Build a foundation for automated observations  
- 🤖 Enable future integrations (imaging systems, etc.)
